## My master's thesis project
Zařízení založené na platformě STM32 které generuje signál DMX512 na základě příchozích MIDI zpráv pomocí převodní tabulky, kterou lze nastavit pomocí [desktopové aplikace pro Windows](https://github.com/PatrikHjdch/scriabinConfigApp).
